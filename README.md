# Zhang CPU Lab Website

This repository contains the source code for the Zhang CPU Lab website.

## Website

https://zhang-cpulab.github.io/zhang-cpulab/

## Main files

- `index.html`: homepage
- `research.html`: research page
- `team.html`: team page
- `publications.html`: publications page
- `join-us.html`: recruitment page
- `news.html`: news page
- `contact.html`: contact page

## Folders

- `assets/css/`: website style files
- `assets/images/`: images used on the website

## How to update

To update the website, edit the corresponding `.html` file and commit the changes.

For example:

- Edit `publications.html` to add new publications.
- Edit `team.html` to add new members.
- Edit `news.html` to add lab news.
- Replace `assets/images/hero.png` to change the homepage background image.
